Here's how to make it predictable and easy.</p><p>Though it's not a big deal, but running Etherpad as hosted on Windows is troublesome. Here's how to make it predictable and easy.</p> <p>What you'd need beforehand:</p> <p><strong>1.</strong> Properly install system-wide <code>Node.js</code>, and of course NPM with it. Get the latest-and-stable <a href="https://nodejs.org/en/download/">from NodeJs.org</a>.<br> After installing, check it from command prompt.</p> <div class="highlight"><pre><span></span>C:\users\alpha\&gt; node -v v5.11.0 C:\users\alpha\&gt; npm verison { npm: &#39;3.8.6&#39;, ares: &#39;1.10.1-DEV&#39;, http_parser: &#39;2.7.0&#39;, icu: &#39;56.1&#39;, modules: &#39;47&#39;, node: &#39;5.11.0&#39;, openssl: &#39;1.0.2g&#39;, uv: &#39;1.8.0&#39;, v8: &#39;4.6.85.31&#39;, zlib: &#39;1.2.8&#39; } </pre></div> <p>Version(s) in your output may not be same but nothing to worry.</p> <p><strong>2.</strong> Download and install latest Python 2.7 from <a href="https://www.python.org/downloads/">Python.org</a>. Please note that <strong>Python2</strong> is needed as system-default, NOT <strong>Python 3</strong>.<br> Check it by running Python interpreter from command-prompt.</p> <div class="highlight"><pre><span></span>C:\users\alpha\&gt; python Python 2.7.11 (v2.7.11:6d1b6a68f775, Dec 5 2015, 20:32:19) [MSC v.1500 32 bit (Intel)] on win32 Type &quot;help&quot;, &quot;copyright&quot;, &quot;credits&quot; or &quot;license&quot; for more information. &gt;&gt;&gt; </pre></div> <p><strong>3.</strong> Now, make sure you have <strong>Microsoft Build Tools 2013</strong> or newer properly installed on you system. Grab it <a href="http://go.microsoft.com/fwlink/?LinkID=320711">from Microsoft</a>.<br> If the given link don't work, look for it under <code>Tools for Visual Studio 2013</code> in <a href="https://www.visualstudio.com/downloads">VisualStudio's download site</a>.</p> <p>Only after you have done these properly, proceed to download <code>Etherpad-lite</code> from <a href="http://etherpad.org/#download">etherpad.org</a> The download size should be over <strong>50MB</strong>. </p> <p>Do <strong>NOT</strong> go for building from the source unless you must.</p> <p><strong>Download</strong> the zip, <strong>unzip</strong> it, <strong>move</strong> it to your desired folder, and then run <code>start.bat</code> in the folder.</p> <p>You will never know how much time i just saved you.</p> <p>You're welcome ... ;)</p>
