Khaled Monsoor :: Blog - centoshttps://blog.kmonsoor.com/2018-07-07T00:00:00+06:00Install latest Python 3 on Linux CentOS 72018-07-07T00:00:00+06:002018-07-07T00:00:00+06:00Khaled Monsoortag:blog.kmonsoor.com,2018-07-07:/install-latest-python3-on-centos-7/<p>Install latest and greatest Python 3 on CentOS 7 systems</p><h2>Why</h2> <p>Not all distro created equal. <br> Some are created to join the space-race, some are to hold unto the leagcy. Some are cutting-edge, some are cutting edge. Some born to boot-up IoT devices, some are to pull up heavy-graphics.</p> <p>That's the fun (albeit, power) of Linux.</p> <p><img alt="An uber, cool image by Ash Edmonds" src="https://i.imgur.com/3lUdFA4.jpg"></p> <blockquote> <p>Photo by <a href="https://unsplash.com/@badashproducts">Ash Edmonds</a> on Unsplash</p> </blockquote> <p>CentOS 7 is a powerful and stable distro which runs on thousands (probably, millions) production-grade servers.<br> In stability, it's a beast. However, it don't ship with Python 3, by default. You can install it via EPEL repository, or this below simple steps.</p> <p>Also, take a note. The Python <strong>2</strong> comes with the system, which is probably 2.7.5, do <strong>NOT</strong> mess with it. Many system components rely on that specific version. If you need later version of 2, use virtualenv or pipenv.</p> <h3>Prepare your system</h3> <p>Start with installing pre-requisite utilities for compilation and development support.</p> <div class="highlight"><pre><span></span>$ sudo yum update <span class="o">&amp;&amp;</span> sudo yum groupinstall -y <span class="s2">&quot;development tools&quot;</span> $ sudo yum install -y zlib-devel bzip2-devel openssl-devel ncurses-devel <span class="se">\</span> sqlite-devel readline-devel tk-devel gdbm-devel <span class="se">\</span> db4-devel libpcap-devel xz-devel expat-devel </pre></div> <h3>Download latest Python source code from Python.org</h3> <div class="highlight"><pre><span></span>$ wget https://www.python.org/ftp/python/3.6.6/Python-3.6.6.tar.xz $ tar xf Python-3.6.6.tar.xz $ <span class="nb">cd</span> Python-3.6.6 </pre></div> <h3>Enable performance optimizations (optional, but highly recommended)</h3> <div class="highlight"><pre><span></span>$ ./configure --prefix<span class="o">=</span>/usr/local --enable-shared <span class="nv">LDFLAGS</span><span class="o">=</span><span class="s2">&quot;-Wl,-rpath /usr/local/lib&quot;</span> $ ./configure --enable-optimizations </pre></div> <h3>Build and install</h3> <div class="highlight"><pre><span></span>$ make $ sudo make altinstall </pre></div> <p>Now, Python 3.6.6 is ready to be used in your system; located in <code>/usr/local/bin/python3.6</code></p> <div class="highlight"><pre><span></span>$ which python3.6 /usr/local/bin/python3.6 $ python3.6 Python <span class="m">3</span>.6.6 <span class="o">(</span>default, Jul <span class="m">10</span> <span class="m">2018</span>, <span class="m">14</span>:04:26<span class="o">)</span> <span class="o">[</span>GCC <span class="m">4</span>.8.5 <span class="m">20150623</span> <span class="o">(</span>Red Hat <span class="m">4</span>.8.5-28<span class="o">)]</span> on linux Type <span class="s2">&quot;help&quot;</span>, <span class="s2">&quot;copyright&quot;</span>, <span class="s2">&quot;credits&quot;</span> or <span class="s2">&quot;license&quot;</span> <span class="k">for</span> more information. &gt;&gt;&gt; </pre></div> <p>For convenience, you can create a symbolic-link with a shorter name. If you had system-installed Python3 (unlikely), <strong>don't</strong> do this, as some system-components may depend on that specific older version of Python 3.</p> <div class="highlight"><pre><span></span>$ sudo ln -s /usr/local/bin/python3.6 /usr/local/bin/python3 $ python3 Python <span class="m">3</span>.6.6 <span class="o">(</span>default, Jul <span class="m">10</span> <span class="m">2018</span>, <span class="m">14</span>:04:26<span class="o">)</span> <span class="o">[</span>GCC <span class="m">4</span>.8.5 <span class="m">20150623</span> <span class="o">(</span>Red Hat <span class="m">4</span>.8.5-28<span class="o">)]</span> on linux Type <span class="s2">&quot;help&quot;</span>, <span class="s2">&quot;copyright&quot;</span>, <span class="s2">&quot;credits&quot;</span> or <span class="s2">&quot;license&quot;</span> <span class="k">for</span> more information. &gt;&gt;&gt; </pre></div> <h2><del>Install wheel and pip</del></h2> <p>You don't need to, because <code>Python 3.6.6</code> includes these necessary tools included.</p> <div class="highlight"><pre><span></span>$ pip3.6 -V pip <span class="m">10</span>.0.1 from /usr/local/lib/python3.6/site-packages/pip <span class="o">(</span>python <span class="m">3</span>.6<span class="o">)</span> $ wheel version wheel <span class="m">0</span>.29.0 </pre></div>
